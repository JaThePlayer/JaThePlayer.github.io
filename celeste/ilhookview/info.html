Celeste IL Hook Viewer

This viewer receives data about all IL hooks applied on all methods, via launching the game, then opening a map with all mods from these gamebanana categories: 
helpers
other/misc
tools:other/misc
Then, the ildiff_all command from Mapping Utils is used to automatically create the database.

Warning

Not all code mods are included in the database yet. Most notably, none of the code mods that ship with maps are included. This may change in the future.
This database may not include all hooks - if a mod lazily loads hooks as they're needed, it's likely that they won't be included.

The database is updated manually.
